Penjelasan Kode HTML

Di bawah ini adalah penjelasan untuk kode HTML yang diberikan:

  • <head>:
    Bagian ini berisi metadata tentang dokumen, yang tidak langsung ditampilkan di halaman web tetapi digunakan oleh mesin pencari, browser, dan perangkat lainnya.

  • <meta charset="UTF-8">:
    Menentukan set karakter yang digunakan dalam halaman.

  • <meta name="viewport" content="width=device-width, initial-scale=1.0">:
    Memberikan instruksi kepada browser tentang cara menampilkan halaman pada perangkat yang berbeda.

  • <title>Document</title>:
    Menentukan judul halaman web.
